Waarom heb ik voor dit product gekozen?

Er smartphone is een handig toestel dat voor veel dingen gebruikt kan worden, je kan er mee bellen, berichten versturen, games spelen, als gps gebruiken en nog veel meer. Een smartphone is een toestel dat men deze dagen absoluut nodig heeft. Sinds je een smartphone ook makkelijk kan indelen in verschillende merken, prijsklasses, kleuren, enzovoort vondt ik het een goed onderwerp voor deze opdracht.

Beschrijving product

Ik verkoop op mijn website smartphones van verschillende merken. Deze merken hebben allemaal goedkopere en duurdere modellen. Je kan op de productpagina dan ook de verschillende modellen bekijken en vergelijken met elkaar.
