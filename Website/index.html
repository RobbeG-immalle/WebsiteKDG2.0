Productinfo

Waarom heb ik voor dit product gekozen?

Een smartphone is een handig toestel dat voor veel dingen gebruikt kan worden: je kan er mee bellen, berichten versturen, games spelen, als gps gebruiken en nog veel meer. Een smartphone is een toestel dat men deze dagen absoluut nodig heeft. Vermits je een smartphone ook makkelijk kan indelen in verschillende merken, prijsklasses, kleuren, enzovoort vond ik het een goed onderwerp voor deze opdracht.

Beschrijving product

Ik verkoop op mijn website smartphones van verschillende merken. Deze merken hebben allemaal goedkopere en duurdere modellen. Je kan op de productpagina dan ook de verschillende modellen bekijken.

Personalisering

Je kan de kleur van je gsm zelf kiezen, en je kan zelfs afwijken van de basiskleuren omdat wij ook werken met custom paintjobs!
